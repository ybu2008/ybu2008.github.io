yum:Failed to set locale, defaulting to C

Published: Thu 21 January 2016
By Zbaihong

In linux.

tags: yum locale

  • 环境：centos 6.6 x86_64

  • 描述：yum使用时报Failed to set locale, defaulting to C，执行locale命令(设置语言环境)：
    '''sh 01.locale：Cannot set LC_CTYPE to default locale： No such file or directory 02.locale：Cannot set LC_MESSAGES to default locale： No such file or directory 03.locale：Cannot set LC_ALL to default locale： No such file or directory '''

  • 解决：
    '''sh echo "export LC_ALL=en_US.UTF-8" >> /etc/profile source /etc/profile '''
