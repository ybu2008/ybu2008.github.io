Python upgrade from 2.6 to 2.7

Published: Sun 10 January 2016
By Zbaihong

In Python.

tags: python

1、下载python2.7包;

2、安装zlib和zlib-devel，如果是ubuntu系统，是zlib1g；

否则在按setuptools时报missing zlib module的错误

3、安装openssl和openssl-devel；

否则通过easy_install时报Unknown type:https或http

4、解压python tar包，执行下述命令：

./configure --prefix=/usr/local/python-2.7.11
make && make install

5、配置环境:

  • profile
vi /etc/profile
PYTHONPATH=/usr/local/python-2.7.11
PATH=$PATH:$PYTHONPATH/bin
export PATH

保存文件后执行下述命令更新配置：

source /etc/profile
  • python命令：
mv /usr/bin/python /usr/bin/python-2.6
python

此时看到python打印的版本号为2.7。

6、安装setuptools

进入setuptools安装目录执行：

python setup.py install

执行下述命令验证是否成功

easy_install virtualenv

如果不成功请检查上述的zlib或openssl。
